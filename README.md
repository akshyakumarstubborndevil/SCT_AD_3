# Stopwatch App

A simple Stopwatch App developed using Android Studio as part of my Android Development Internship at SkillCraft Technology.

* Start Timer
* Pause Timer
* Reset Timer
* Displays Minutes, Seconds, and Milliseconds
* Simple and User-Friendly Interface

* Java
* XML
* Android Emulator
